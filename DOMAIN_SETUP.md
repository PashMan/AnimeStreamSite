# Подключение домена и настройка почты

Чтобы ваш сайт открывался по красивому адресу (например, `kamianime.club`) и письма приходили без красной плашки "Потенциально опасное письмо", нужно выполнить следующие шаги.

## Часть 1: Подключение домена к Vercel (чтобы сайт открывался)

1.  Зайдите в панель управления **Vercel**.
2.  Перейдите в ваш проект -> **Settings** -> **Domains**.
3.  Введите ваш купленный домен (например, `kamianime.club`) и нажмите **Add**.
4.  Vercel покажет вам необходимые DNS-записи. Обычно это:
    *   **Тип:** `A`
    *   **Имя (Name):** `@`
    *   **Значение (Value):** `76.76.21.21`
    *   *(Или CNAME для поддоменов www)*

5.  **Теперь идем на Reg.ru:**
    *   Зайдите в настройки вашего домена.
    *   Найдите раздел **DNS-серверы и управление зоной**.
    *   Добавьте запись, которую выдал Vercel (A-запись).
    *   *Важно:* Обновление DNS может занять от 15 минут до 24 часов.

---

## Часть 2: Настройка почты (чтобы убрать красную плашку)

Красная плашка появляется, потому что Supabase отправляет письма от своего имени, но подписывается вашим адресом. Чтобы это исправить, нужно использовать свой домен для отправки.

Лучший бесплатный способ — сервис **Resend** (до 3000 писем в месяц бесплатно).

### Шаг 1: Регистрация в Resend
1.  Зайдите на [resend.com](https://resend.com) и зарегистрируйтесь.
2.  В панели нажмите **Add Domain**.
3.  Введите ваш домен (например, `kamianime.club`).
4.  Выберите регион (обычно US или EU, не принципиально).
5.  Resend выдаст вам список DNS-записей (MX, TXT, CNAME).

### Шаг 2: Настройка DNS на Reg.ru
1.  Вернитесь в управление зоной на **Reg.ru**.
2.  Добавьте **ВСЕ** записи, которые показал Resend.
    *   Особенно важны записи типа `TXT` (SPF и DKIM) — именно они говорят почтовым сервисам (Gmail, Mail.ru), что вы — настоящий владелец домена.
    *   Запись `MX` нужна, если вы хотите *получать* ответы на этот домен (не обязательно для рассылки, но желательно).
3.  В Resend нажмите кнопку **Verify DNS Records**. Когда все загорится зеленым ("Verified"), можно продолжать.

### Шаг 3: Получение API ключа
1.  В Resend перейдите в **API Keys**.
2.  Создайте новый ключ (Create API Key), дайте ему имя (например, "Supabase").
3.  **Скопируйте ключ** (он начинается на `re_...`).

### Шаг 4: Подключение к Supabase
1.  Зайдите в панель управления **Supabase**.
2.  Перейдите в **Project Settings** -> **Authentication** -> **Email Settings** (или SMTP Settings).
3.  Включите **Enable Custom SMTP**.
4.  Заполните поля данными от Resend:
    *   **Sender Email:** `support@kamianime.club` (или любое имя на вашем домене)
    *   **Sender Name:** KamiAnime
    *   **Host:** `smtp.resend.com`
    *   **Port:** `465`
    *   **Encryption:** `SSL` (или TLS)
    *   **Username:** `resend`
    *   **Password:** *Вставьте ваш API ключ от Resend (re_...)*
5.  Нажмите **Save**.

Теперь все письма (сброс пароля, подтверждение почты) будут уходить через ваш домен, и красная плашка исчезнет.
